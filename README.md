# 🃏 Blackjack Game (Python CLI)

## 📌 Overview

A command-line Blackjack game built in Python where the player plays against a computer dealer.

## 🎮 Features

* Random card dealing
* Blackjack detection
* Dynamic Ace value handling (1 or 11)
* Dealer AI (draws until 17)
* Input validation
* Replay functionality

## ▶️ How to Run

```bash
git clone https://github.com/Astitvaa27/blackjack-game.git
cd blackjack-game
python main.py
```
